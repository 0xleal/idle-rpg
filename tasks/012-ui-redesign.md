# Task 012: UI Redesign with Frontend Design Plugin

## Goal
Redesign the entire application UI using the frontend-design plugin to create a distinctive, polished, production-grade interface that avoids generic AI aesthetics.

## Current State
- Basic Tailwind CSS styling
- Functional but plain UI
- Emoji-based icons
- Simple color scheme (zinc, skill-specific accent colors)

## Deliverables

### 1. Design System
- Establish a cohesive visual identity
- Custom color palette fitting the RPG theme
- Typography hierarchy
- Consistent spacing and component sizing
- Dark mode as primary (optional light mode)

### 2. Layout Redesign
- Modern sidebar navigation with icons
- Responsive layout for mobile/tablet/desktop
- Better use of screen real estate
- Header with player info (combat level, gold, etc.)

### 3. Skill Pages
- Unified skill page template
- Progress bars with visual flair
- Action cards with better visual hierarchy
- Active action indicator with animation
- Input requirements display

### 4. Inventory Page
- Grid-based item display
- Item tooltips on hover
- Category filtering/tabs
- Stack count display
- Sell/use actions

### 5. Equipment Page
- Paper doll or slot-based layout
- Visual distinction between empty/filled slots
- Equipment comparison on hover
- Stats summary panel

### 6. Combat Page
- Engaging combat UI
- Animated health bars
- Hit splat animations
- Monster display with character art placeholder
- Loot drops with visual feedback
- Combat log

### 7. Dashboard
- Overview of all skills with levels
- Current action status
- Quick stats summary
- Recent achievements/milestones

### 8. Components to Create
- Custom Button variants
- Card component
- Progress bar component
- Tooltip component
- Modal/Dialog component
- Toast notifications
- Loading states

## Design Considerations
- RPG/fantasy theme without being clich√©
- Clean, modern aesthetic
- Smooth animations and transitions
- Accessibility (contrast, focus states)
- Performance (no heavy assets for MVP)

## Acceptance Criteria
- [ ] Cohesive design system established
- [ ] All pages redesigned with new components
- [ ] Responsive on mobile, tablet, desktop
- [ ] Dark mode fully supported
- [ ] Smooth animations and transitions
- [ ] Passes basic accessibility checks
